What is Semester GPA?

Learn How To Calculate Semester GPA

Semester GPA is calculated by multiplying the number of credit hours assigned to a course by the value of the grade earned in that semester A.

How to calculate semester GPA?

Letter grades are each assigned a point value:


To calculate your grade point average, first multiply the number of credits each class is worth by the point value for the letter grade that you earned in that class.
